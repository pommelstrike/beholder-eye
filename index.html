pommelstrike's BeholderEye: Fix your materialbank .lsf files for console compatibility.

Materialbank parser for BG3 .lsx source files to redirect MOD KIT generated project shaders to pre-packaged BG3 game shaders for Game Console compatibility.

Pull the .lsf files into LSLIB's "LSF to LSX" tab from "Baldurs Gate 3\Data\Public\PROJECTNAME_UUID\Content\[PAK]_\PROJECTNAME>".

Note: This tool processes files locally within your browser. Nothing is ever uploaded to any server.
