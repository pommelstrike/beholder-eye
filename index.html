LSX File Processor

A tool to parse and process .lsx files for BG3 console compatibility.

Materialbank parser for BG3 .lsx source files to redirect MOD KIT generated project shaders to pre-packaged BG3 game shaders for Game Console compatibility.

Pull the .lsf files into LSLIB's "LSF to LSX" tab from Data\Public<ModName_UUID>\Assets. Do not use BG3 Multitool. Use the .lsf files from the Project folders as listed above. The one piece is real.
